```

（可选）# 第一次下载项目（远程->本地）
（可选）git clone git@xxxx.git

（可选）# 初始化git项目
（可选）git init

（可选）# 关联git仓库
（可选）git remote add origin git项目地址 (SSH连接后不需要执行该命令)

# 添加 
git add .

# 评论注释
git commit -m "update" （"update"是评论内容）

（可选）# 下拉代码，将远程最新的代码先跟你本地的代码合并一下，如果确定远程没有更新，可以不用。避免修改冲突。
（可选）git pull origin main 

# 存入仓库
git push origin main  （main是分支名称）
```
